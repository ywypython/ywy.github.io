基础数据类型

str:

  • capitalize 首字母大写,其余小写
  • swapcase 大小写反转
  • title 每个单词首字母大写
  • center 居中
  • find 通过元素找索引,找到第一个就返回,找不到返回-1
  • index 通过元素找索引,找到第一个就返回,找不到就报错

tuple :

  • 元祖中如果只有一个元素,并且没有逗号,那么他不是元祖,他与该元素数据类型一样
  • count 计数

list:

循环一个列表时,最好不要改变列表的大小,这样会影响你最终的结果

  • index 通过元素找索引
  • count 计数
  • sort 排序
  • reverse 列表反转
  • 列表可以相加
  • 列表可以与数字相乘

dict:

  • update 更新,有就更新,没有就添加
  • fromkeys 以其中的第一个元素作为字典的键,第二个元素作为字典的值

数据类型之间的转换

编码

  • encode(‘编码类型’) 编码
  • decode(‘解码类型’) 解码

python中所有的数据类型,除了bytes都是unicode编码储存,

id == is

  • id : 获取对象内存地址
  • is : 比较内存地址是否相同
  • == : 比较值是否相同

代码块

  • 一个文件,交互式命令一行就是一个代码块.

统一代码块Ixia缓存机制 (字符串驻留机制)

  • 所有数字,bool 几乎所有字符串
  • 优点 : 提升性能,节省内存空间

不同代码块的缓存机制 (小数据池)

  • 在内存中开辟两个空间,一个存储-5~256的int,一个空间存储一定规则的字符串,如果你的代码中遇到了满足条件的数据,直接引用提前创建的.
  • -5~256 int, bool , 满足一定规则的字符串
  • 优点 : 提升性能,节省内存空间

集合

  • 列表去重
  • 关系测试 (交集,并集差集)

深浅拷贝:

  • 浅拷贝:
    • 在内存中开辟一个新的空间存放拷贝的对象(列表,字典),但是里面的所有元素与被拷贝对象里面的元素公用同一个
