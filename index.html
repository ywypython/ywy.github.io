2.今日内容

  1. 函数的初识

    • 以功能（完成一件事）为导向，一个函数就是一个功能。随调随用
    • 减少代码的重复性，增强代码的可读性。

  2. 函数的结构与调用

    • def meet():
    print("拿出手机")

    print("打开陌陌")

    print('左滑一下')

    print('右滑一下')

    print("找个漂亮的妹子")

    print("问她,约不约啊!")

    print("ok 走起")

'''
结构：def关键字，定义函数。
meet 慢书名：与变量设置相同，具有可描述性。
函数体：速进。函数中尽量不要出现print。

'''

    • return

      • 在函数中遇到 return 直接结束函数。

      • 将数据返回给函数的执行者（调用者 meet()）。

      • 可以设置多个返回值。(类型为元组)

  3. 函数的参数

    1. 函数的传参：让函数封装的这个功能，盘活。

      • 实参 函数的执行传的参数：实际参数
      • 形参 接受的参数形式参数

    2. 从实参角度：

      1. 位置参数 从左至右，一一对应。

    ​ 2.关键字参数 一一对应

    ​ 3.混合传参（位置参数一定要在关键字参数前面）

​ 4.小结：

​ 1.位置参数，按照顺序，一一对应

​ 2.关键字参数，一一对应

​ 3.混合参数：位置参数一定要在关键字参数前面

​ 2.形参角度

​ 1.位置参数 与实参角度的位置参数是一种

​ 2.默认值参数 （意义：普遍经常使用）

​ 3.函数的结构，函数的执行。
